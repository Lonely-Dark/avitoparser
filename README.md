# Simple library to parse the Avito website
## Usage:

Create environment:
```
python -m venv venv
```

On Windows:
```
venv\Scripts\activate.bat
```

On Linux, or MacOS:
```
source venv/bin/activate
```

Install dependencies:
```
pip install -r requirements.txt
```

Import a library:
```
from avitoparse import AvitoParser
```

Create an object instance:
```
parser = AvitoParser()
```
